# Yahoo-TW-ETF-Scraper

這是一個專為台灣投資者設計的全自動 ETF 數據抓取工具。只需點擊一下，即可從 Yahoo 股市抓取 330+ 檔台股 ETF 的詳細數據，並自動整合 1 週與 1 個月的績效表現。

## 🌟 特點
- **全自動化**：使用 Selenium 模擬瀏覽器捲動，自動載入所有資料。
- **背景執行**：支援 Headless 模式，執行時不會彈出瀏覽器視窗。
- **數據完整**：整合搜尋頁面（資產、管理費等）與績效頁面（短期績效）。
- **簡單易用**：為長輩設計的 `.bat` 一鍵執行檔。

## 📊 抓取欄位
- 股票名稱 / 代號
- 股價 / 漲跌幅
- **短期績效**：1週、1個月
- **中長期績效**：年初至今、3個月、6個月、1年、2年、3年、5年、10年
- **詳細數據**：成交量、殖利率、管理費、資產規模、成立年數

## 🛠️ 安裝說明
1. 確保電腦已安裝 [Python](https://www.python.org/)。
2. 安裝必要套件：
   ```bash
   pip install selenium webdriver-manager pandas openpyxl
   ```

## 🚀 使用方法
- **方法一**：連按兩下執行 `全自動抓取ETF.bat`。
- **方法二**：在終端機執行 `python selenium_get_etf.py`。
